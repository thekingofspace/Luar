MonoBehaviour

LUAR includes a built-in base class, MonoBehaviour, for an entirely class-based, Unity-style model. Subclass it and implement any of the lifecycle hooks, then call run to drive them.

HookWhen it runs
Awake()Once, before Start, when the behaviour is created.
Start()Once, after Awake.
Update()Once per frame.
OnDestroy()Once, at the end.
class Spinner extends MonoBehaviour {
  public angle: number = 0
  function Start(): void print("start") end
  function Update(): void self.angle += 90 end
  function OnDestroy(): void print("done") end
}

run(2)                -- Start, Update x2, OnDestroy on every behaviour
local one = spawn(Spinner)   -- construct one and run Awake + Start
  • run(frames) constructs one instance of every concrete MonoBehaviour subclass, calls Awake then Start, runs Update the given number of frames (default 1), then OnDestroy.
  • spawn(Class, ...args) constructs a single behaviour, runs its Awake and Start, and returns the instance.
