Getting started

A LUAR source file uses the .luar extension.

print("hello, world")

Running scripts

The runner executes a script after checking it with Ferrite.

run hello.luar          # run a source file
run hello.luarc         # run a precompiled file
compile hello.luar hello.luarc   # precompile to .luarc

File types

ExtensionMeaning
.luarA LUAR source file.
.luarcA precompiled file (a serialized program). See Precompilation.
