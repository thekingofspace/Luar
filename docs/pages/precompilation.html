Precompilation

A program can be precompiled to a compact binary that loads and runs on the full interpreter, so every language feature works from precompiled code.

compile script.luar script.luarc   # produce a .luarc
run script.luarc                   # run it

From Rust:

let bytes = luar::precompile_source(src)?;   // serialize
let interp = luar::run_precompiled(&bytes)?;  // load and run

A .luarc file is the parsed, type-scrubbed program. Because it runs on the interpreter, classes, enums, mixins, operators, MonoBehaviour, and the standard library all behave exactly as they do from source.
