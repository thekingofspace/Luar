Control flow

if / elseif / else

if score > 90 then
  print("A")
elseif score > 80 then
  print("B")
else
  print("C")
end

Every value is truthy except false and nil.

while and for

while n > 0 do
  n -= 1
end

for i = 1, 10 do print(i) end
for i = 10, 1, -1 do print(i) end   -- explicit step

for index, value in ipairs(list) do print(index, value) end
for key, value in pairs(map) do print(key, value) end

Name an unused loop variable _ to silence UnusedLoopVariable. break exits the innermost loop.

switch

switch is an expression. The first case whose value equals the subject runs, and a return inside it becomes the value of the switch. An optional default runs when no case matches. Each case has its own end, and the whole switch is closed by a final end.

local label = switch(state)
  case "idle"
    return "waiting"
  end
  case "run"
    return "working"
  end
  default
    return "unknown"
  end
end
