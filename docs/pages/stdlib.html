Standard library

The standard library is a set of ready-made functions grouped into namespaces: string, table, math, bit32, os, and coroutine. Each is a global table; you call a function with a dot, like string.upper("hi"). Every function is listed below with what it does. Positions in strings and arrays are 1-based: the first item is at index 1.

string

Functions for working with text.

FunctionDescription
string.len(s)The number of characters in s. Same as #s.
string.sub(s, i, j)The part of s from position i to j (inclusive). j may be omitted to go to the end.
string.upper(s)A copy of s with letters uppercased.
string.lower(s)A copy of s with letters lowercased.
string.rep(s, n)The string s repeated n times.
string.reverse(s)The characters of s in reverse order.
string.byte(s, i)The numeric code of the character at position i.
string.char(...)A string built from the given character codes.
string.find(s, sub)The start and end positions of the first occurrence of sub in s, or nil if not found.
string.contains(s, sub)True if s contains sub.
string.startswith(s, prefix)True if s begins with prefix.
string.endswith(s, suffix)True if s ends with suffix.
string.trim(s)A copy of s with leading and trailing whitespace removed.
string.split(s, sep)An array of the pieces of s split on each occurrence of sep.
string.format(fmt, ...)A formatted string. fmt contains placeholders such as %s (string), %d (integer), %f or %.2f (float), and %x (hex), filled in by the remaining arguments.
print(string.upper("luar"))                 -- LUAR
print(string.sub("hello", 1, 3))           -- hel
print(string.split("a,b,c", ","))            -- { "a", "b", "c" }
print(string.format("%s is %d", "x", 42))   -- x is 42

table

Functions for working with tables used as arrays.

FunctionDescription
table.insert(t, value)Append value to the end of the array.
table.insert(t, pos, value)Insert value at position pos, shifting later items up.
table.remove(t)Remove and return the last element.
table.remove(t, pos)Remove and return the element at pos, shifting later items down.
table.concat(t, sep)Join an array of strings into one string, separated by sep.
table.sort(t)Sort the array in place in ascending order.
table.sort(t, compare)Sort using a comparator function; compare(a, b) returns true if a should come before b.
table.keys(t)An array of the table's keys.
table.pack(...)Collect the given values into a new array.
table.unpack(t)Expand an array into multiple return values.
local t = { 3, 1, 2 }
table.insert(t, 4)
table.sort(t)
print(table.concat(t, ", "))   -- 1, 2, 3, 4
table.sort(t, function(a, b) return a > b end)   -- descending

math

Numeric functions and constants.

FunctionDescription
math.abs(x)The absolute value of x.
math.ceil(x)The smallest integer not less than x (round up).
math.floor(x)The largest integer not greater than x (round down).
math.round(x)x rounded to the nearest integer.
math.sqrt(x)The square root of x.
math.sin(x), math.cos(x), math.tan(x)Trigonometric functions; x is in radians.
math.asin(x), math.acos(x), math.atan(x)Inverse trigonometric functions.
math.exp(x)e raised to the power x.
math.log(x)The natural logarithm of x.
math.pow(x, y)x raised to the power y (same as x ^ y).
math.fmod(x, y)The remainder of dividing x by y.
math.modf(x)The integer and fractional parts of x.
math.max(...)The largest of the given numbers.
math.min(...)The smallest of the given numbers.
math.clamp(x, lo, hi)x limited to the range lo to hi.
math.sign(x)-1, 0, or 1 depending on the sign of x.
math.deg(x)Radians converted to degrees.
math.rad(x)Degrees converted to radians.
math.random()A random float in the range 0 (inclusive) to 1 (exclusive).
math.random(m)A random integer from 1 to m.
math.random(lo, hi)A random integer from lo to hi.
math.randomseed(n)Seed the random generator so results are repeatable.
ConstantValue
math.piThe ratio pi.
math.hugePositive infinity.
math.maxinteger / math.minintegerThe largest and smallest integers.

bit32

Bitwise operations on integers, treated as 32-bit.

FunctionDescription
bit32.band(a, b, ...)Bitwise AND of the arguments.
bit32.bor(a, b, ...)Bitwise OR.
bit32.bxor(a, b, ...)Bitwise exclusive OR.
bit32.bnot(a)Bitwise NOT (flip every bit).
bit32.lshift(a, n)Shift a left by n bits.
bit32.rshift(a, n)Shift a right by n bits (zero fill).
bit32.arshift(a, n)Arithmetic right shift (sign-preserving).

os

FunctionDescription
os.time()The current time, in seconds.
os.clock()A high-resolution timer, useful for measuring how long something takes.

coroutine

A coroutine is a function that can pause itself and be resumed later, keeping its place and its local variables. It is a way to write code that yields control and continues where it left off, without threads.

FunctionDescription
coroutine.create(fn)Make a new coroutine that will run fn when first resumed.
coroutine.resume(co, ...)Start or continue a coroutine. The arguments are passed to the function on the first resume, or become the result of the matching yield after that. Returns whether it succeeded plus any yielded or returned values.
coroutine.yield(...)Pause the current coroutine, handing the given values back to whoever resumed it.
coroutine.status(co)The coroutine's state, such as suspended, running, or dead.
coroutine.close(co)Finish a suspended coroutine so it can be cleaned up.
local co = coroutine.create(function(a)
  print("start", a)
  local b = coroutine.yield(a + 1)   -- pause, hand back a+1
  print("resumed with", b)
end)
coroutine.resume(co, 10)   -- start: 10   then yields 11
coroutine.resume(co, 20)   -- resumed with: 20

Global builtins

Besides the namespaces above, there are functions available directly without a namespace, such as print, type, pcall, pairs, and the reflection functions. See Builtins for the full list, and for how to override them.
