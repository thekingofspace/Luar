Closures

local make = function(start)
  local n = start
  return function()
    n += 1
    return n
  end
end

local next = make(10)
print(next())   -- 11
print(next())   -- 12

Multiple returns and varargs

function minmax(a, b)
  if a < b then return a, b end
  return b, a
end

function sum(...)
  local total = 0
  for _, n in ipairs({ ... }) do total += n end
  return total
end

Call sugar

When a function takes a single string or table literal, the parentheses are optional.

print "hello"              -- print("hello")
configure { debug = true }   -- configure({ debug = true })

Generic parameters

local function identity<T>(x)
  return x
end

Generic parameters are used by the editor and scrubbed before execution.

Documentation comments

A comment directly above a function documents it. Markdown such as **bold** is rendered in the hover box, and lines beginning with @ are highlighted as tags.

-- Clamp a number into the range [lo, hi].
-- @param value the number to clamp
-- @return number
function clamp(value, lo, hi)
  if value < lo then return lo end
  if value > hi then return hi end
  return value
end
